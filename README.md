# 交互式数据可视化平台

一个基于Plotly.js和D3.js构建的现代化数据可视化Web应用，完全实现了第7章中的所有可视化功能。

## 🌟 功能特性

### 3D图表可视化
- **3D散点图**: 支持自定义数据点数量、范围和颜色方案
- **3D曲面图**: 生成数学函数的3D曲面表示
- **3D线框图**: 展示数据的线框结构
- **自定义数据输入**: 支持CSV格式的用户数据导入

### 动画功能
- **正弦波动画**: 实时演示正弦函数的动态变化
- **3D闪烁星星**: 模拟星空的闪烁效果
- **螺旋动画**: 3D螺旋线的动态旋转
- **动画控制**: 开始、暂停、停止、重置等完整控制
- **参数调节**: 实时调整动画速度和振幅

### 地图可视化
- **散点地图**: 美国城镇人口分布的点状展示
- **气泡地图**: 根据人口大小显示不同尺寸的气泡
- **数据筛选**: 按人口数量和城市数量筛选
- **统计信息**: 实时显示数据统计
- **配色主题**: 多种颜色方案可选

### 交互功能
- **缩放平移**: 所有图表支持鼠标缩放和拖拽
- **悬停提示**: 鼠标悬停显示详细数据信息
- **全屏模式**: 支持图表全屏展示
- **图表下载**: 一键下载PNG格式图片
- **键盘快捷键**: 支持ESC退出全屏，Ctrl+S下载等

### 响应式设计
- **移动端适配**: 完美支持手机和平板设备
- **触控优化**: 优化触控操作体验
- **自适应布局**: 根据屏幕尺寸自动调整
- **性能优化**: 流畅的动画和交互体验

## 🎯 核心算法

### 3D数据生成
```javascript
// 生成随机3D数据点
function generate3DData(points, minRange, maxRange) {
    const x = [], y = [], z = [];
    for (let i = 0; i < points; i++) {
        x.push(Math.random() * (maxRange - minRange) + minRange);
        y.push(Math.random() * (maxRange - minRange) + minRange);
        z.push(Math.random() * (maxRange - minRange) + minRange);
    }
    return { x, y, z };
}
```

### 动画循环
```javascript
// 动画主循环
function animate() {
    updateFrame();
    render();
    animationId = setTimeout(() => animate(), speed);
}
```

### 地图数据处理
```javascript
// 数据筛选和统计
function filterAndProcessData(data, minPop, count) {
    return data
        .filter(city => city.pop >= minPop)
        .slice(0, count)
        .sort((a, b) => b.pop - a.pop);
}
```

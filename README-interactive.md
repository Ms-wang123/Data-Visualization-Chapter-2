# 交互式数据可视化仪表板 - 第2章

这是一个基于第2章数据可视化内容的交互式仪表板，展示了10种不同类型的图表，具有丰富的交互功能和现代化的用户界面。

## 🌟 主要功能

### 1. 动态数据可视化组件
- **10种图表类型**: 折线图、柱形图、条形图、堆积面积图、直方图、饼图、散点图、箱形图、雷达图、误差棒图
- **实时参数更新**: 所有图表参数可实时调整，立即反映在可视化效果中
- **响应式设计**: 自动适配不同屏幕尺寸，支持桌面和移动设备

### 2. 交互式图表元素
- **悬停显示详情**: 鼠标悬停在图表元素上显示详细数据信息
- **点击筛选功能**: 支持通过点击图表元素进行数据筛选
- **工具提示**: 智能工具提示系统，显示精确的数据值

### 3. 用户操作面板
- **样式控制**: 调整颜色主题、点大小、线条样式等
- **动画设置**: 自定义动画速度和效果
- **显示选项**: 控制图例、网格线、平滑曲线等显示元素

### 4. 移动端适配
- **触屏交互**: 完全支持触摸屏设备的交互操作
- **响应式布局**: 在不同设备上自动调整布局和控件大小
- **手势支持**: 支持缩放、平移等手势操作

### 5. 案例展示区
- **实际应用场景**: 4个真实业务场景的案例展示
- **一键加载**: 点击案例卡片即可加载对应的图表配置
- **场景说明**: 每个案例都有详细的应用场景描述

## 🚀 快速开始

### 直接访问
打开 `interactive-dashboard.html` 文件即可在浏览器中运行。

### 功能使用
1. **选择图表类型**: 在左侧控制面板选择不同的图表类型
2. **调整参数**: 使用滑块和选项调整图表样式和参数
3. **交互操作**: 鼠标悬停查看详情，点击图表元素进行筛选
4. **数据管理**: 支持添加/删除数据行，导入导出数据
5. **案例学习**: 浏览案例展示区，了解实际应用场景

## 📊 支持的图表类型

| 图表类型 | 描述 | 适用场景 |
|---------|------|----------|
| 折线图 | 展示数据随时间变化的趋势 | 温度变化、销售趋势 |
| 柱形图 | 比较不同类别的数值大小 | 财务数据、业绩对比 |
| 条形图 | 水平方向的柱形图 | 替代率、排名数据 |
| 堆积面积图 | 展示多个数据系列的累积效果 | 费用统计、资源分配 |
| 直方图 | 展示数据分布情况 | 分数分布、频率统计 |
| 饼图 | 展示各部分占整体的比例 | 消费占比、市场份额 |
| 散点图 | 展示两个变量之间的关系 | 相关性分析、分布模式 |
| 箱形图 | 展示数据的分布和异常值 | 统计分布、数据质量 |
| 雷达图 | 多维度数据比较 | 能力评估、性能对比 |
| 误差棒图 | 展示数据的不确定性 | 实验数据、测量误差 |

## 🎨 自定义功能

### 颜色主题
- 6种预设颜色主题：默认、商务、自然、热情、优雅、活力
- 实时切换，立即生效

### 动画效果
- 可调节动画速度（100ms - 2000ms）
- 平滑线条选项
- 悬停放大效果

### 数据操作
- **实时编辑**: 直接在数据表格中修改数值
- **添加/删除**: 动态管理数据行
- **导入导出**: 支持JSON格式的数据导入导出

## 📱 移动端特性

- **触摸优化**: 所有控件都针对触摸屏优化
- **手势支持**: 支持缩放、平移等手势
- **响应式布局**: 在小屏幕上自动调整控件布局
- **全屏模式**: 支持图表全屏显示

## 🔧 技术栈

- **前端框架**: Vue.js 3.2.47
- **图表库**: Chart.js
- **样式**: 纯CSS3，支持现代浏览器特性
- **交互**: 原生JavaScript事件处理
